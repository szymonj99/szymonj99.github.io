What is a computer?

A computer is an electronic device, which consists of multiple components. It can process (change information), follow instructions (what you tell it to do), and show you the outcome (result).

What components does a computer have?

A computer has a Central Processing Unit (CPU), a Motherboard (a circuit board where everything is connected), Random Access Memory (RAM), a storage drive, preipherals (mouse, keyboard, printer, monitor) and a graphics module (GPU, or iGPU).

How does a computer work? - Advanced

A computer processes data using electricity, and the instructions the processor receives from other components. The electricity can switch a transistor state inside the CPU to either a 0, or a 1. These numbers represent the state of the transistor. A modern computer has billions of transistors, each of which can switch states millions of times per second. This results in very fast processing speeds.

Components Breakdown

Various computer components have various functions and uses.

  • The CPU is responsible for memory managment, processing instructions and managing input and output from the computer.
  • The RAM stores software information which can be accessed quickly. This information is cleared when the device is restarted (RAM is called volatile storage).
  • The Motherboard acts as the connecting medium for all the components. It provides electricity to the components.
  • Storage Disks such as a Hard Disk Drive, or a Solid State Drive is called persistent, as it remains stored after the device shuts down. These disks store information which the user can access later.
  • The GPU is responsible for showing you the images you see on the screen. It processes images VERY fast!
  • Peripherals such as a mouse, keyboard (and even your fingers!) are used as a medium between the user and the device. They allow the user to interact with their device.
